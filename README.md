## Technologies and new concepts updated.

* Java 8.
* Spring-Boot 2.
* Spring Data JPA.
* Spring Security.
* Spring REST.
* Spring MVC.
* Logger with LogBack Appender RollingFile.
* Swagger UI.
* OpenAPI v3.
* JSON Web Token (JWT).
* Password Encryption.
* Mockito Testing.
* JUnit Testing.
* Integration Testing.
* Hypermedia Driven RESTful.
* Paging and Sorting.
* Securing APIs.
* Controlling APIs exposure.
* Docker Image.
* Docker Containers.
* Docker Containers Databases.
* Docker Migration Database with Flyway.
* Docker Hub.
